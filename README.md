# Hangman Game

A hangman game, guess the word by pressing a keyboard key.



### Tech stack

* HTML.
* CSS.
* JavaScript.



### **Features**

* Begin typing to enter your guesses.

* How many guesses left.

* Reset game with guesses.

* You will see a message with the right guess if you failed.
